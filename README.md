# 🧠 AI Resume Screening System

## 📌 Overview
An AI-based system that compares resumes with job descriptions and calculates a match score using NLP techniques.

---

## 🚀 Features
- Resume parsing (PDF)
- Skill extraction
- Match score using TF-IDF & cosine similarity
- Missing skills detection
- Streamlit web app interface

---

## 🛠️ Tech Stack
- Python  
- spaCy (NLP)  
- Scikit-learn  
- Streamlit  

---

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
